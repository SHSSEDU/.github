# 山河大学
山河大学（英语：Shanhe University，缩写：SHSSEDU）。 全国首个大学生虚拟自办大学。 山河大学并不是一所实际意义上的大学，而是由山河四省（河北、河南、山东、山西）的学子们汇聚意念共同形成的一种意志和向往，起源于2023年左右，该校因河北、河南、山东、山西四省高等教育资源不足而建立。   
注：为什么不叫SHU呢？因为SHU是上海大学。所以采用“山河四省教育”，简称SHSSEDU
